# Student Efficiency Analysis through ETL Process

## Project Overview
This project analyzes student efficiency and academic performance using an ETL workflow combined with data visualization techniques. The goal was to transform raw student-related data into structured datasets suitable for analysis and to identify factors influencing academic outcomes.

## Tools and Technologies
- Python (NumPy, Pandas, SciPy)
- SQL
- ETL concepts
- Microsoft Power BI

## Project Workflow
- Extracted raw student-related data from source files
- Cleaned and transformed data using Python
- Structured datasets for analytical processing
- Applied statistical analysis to identify patterns in student habits and academic performance
- Created visual reports and dashboards in Power BI for result interpretation

## Key Objectives
- Explore relationships between student behavior and academic efficiency
- Prepare data for analytical reporting
- Present findings through visual dashboards

## Outputs
- Cleaned analytical dataset
- Processed data tables
- Power BI visualizations and reports
